# "GAME DAY TOURS" – Data Analysis Project

## Overview
"GAME DAY TOURS" is a data analysis project focused on improving decision-making for a tourism operator in Guatemala through the integration of structured and unstructured data. The project implements a hybrid data architecture combining SQL Server and MongoDB, along with analytical processing using Python and R, and business intelligence dashboards in Power BI.

The goal of the system is to optimize operational efficiency, analyze customer behavior, and generate actionable insights through data.

---

## Objectives
- Analyze customer behavior and tourism demand patterns  
- Monitor key business metrics (KPIs) such as revenue, reservations, and customer satisfaction  
- Support decision-making through interactive dashboards  
- Integrate multiple data sources into a unified analytical workflow  

---

## Data Architecture
The project uses a hybrid architecture:
  - **SQL Server** → Structured transactional data  
    - Customers  
    - Reservations  
    - Payments  
    - Logistics  

  - **MongoDB** → Semi-structured data  
    - Customer reviews  
    - Dynamic content  

---

## Technologies Used
- **SQL Server** – Relational database design and queries  
- **MongoDB** – NoSQL database for flexible data  
- **Python** – Data processing and analysis (pandas, numpy, matplotlib)
